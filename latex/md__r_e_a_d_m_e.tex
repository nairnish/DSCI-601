This is the training and test data for the Discriminating between Similar Languages (DSL) task at Var\+Dial 2017.\hypertarget{md__r_e_a_d_m_e_autotoc_md9}{}\doxyparagraph{The package contains the following files\+:}\label{md__r_e_a_d_m_e_autotoc_md9}

\begin{DoxyEnumerate}
\item DSL-\/\+TRAIN.\+txt -\/ Training set for the DSL task
\item DSL-\/\+DEV.\+txt -\/ Development set for the DSL task
\item DSL-\/\+DEV.\+txt -\/ Unlabelled test set
\item DSL-\/\+TEST-\/\+UNLABELLED.\+txt -\/ Test set with gold labels
\item README.\+txt -\/ Brief description of the DSL data
\end{DoxyEnumerate}

Each line in the .txt files are tab-\/delimited in the format\+: sentence$<$tab$>$language-\/label

For more details (like data stats) you can refer to the Var\+Dial 2017 task paper\+:
